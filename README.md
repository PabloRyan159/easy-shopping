# 🌍 Easy Shopping

Projeto desenvolvido durante meus estudos de **HTML5** e **CSS3**.

O objetivo foi reproduzir um layout moderno e totalmente responsivo, praticando posicionamento de elementos, Flexbox e adaptação para diferentes tamanhos de tela.

<h2>Esse foi um projeto que desenvolvi durante os estudos no <a href="https://www.devclub.com.br">DevClub</a>.</h2>
